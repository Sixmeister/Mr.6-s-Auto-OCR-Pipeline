## Appendix 1 Code Overview

This folder is prepared for the thesis appendix section "附录1 核心代码文件说明".

The full project source files remain in the repository root and the `test/` folder.
This directory does not duplicate those scripts. Instead, it provides an index so a
reader can quickly locate the code files that correspond to the implementation stages
described in Chapter 4 of the thesis.

Recommended reading order:

1. `test/test_ocr_and_codes_final.py`
2. `auto_ocr_pipeline_v0.1.py`
3. `auto_ocr_pipeline_v0.2gui.py`
4. `auto_ocr_pipeline_v0.4.py`
5. `auto_ocr_pipeline_v0.5.py`
6. `auto_ocr_pipeline_v0.63.py`
7. `auto_ocr_pipeline_v0.71.py`

See `code_file_list.md` for the detailed mapping between thesis sections and code files.
