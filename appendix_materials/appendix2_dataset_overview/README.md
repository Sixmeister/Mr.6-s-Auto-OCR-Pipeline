## Appendix 2 Dataset Overview

This folder is prepared for the thesis appendix section "附录2 标签检测数据集样例与标注示例".

The complete dataset files are stored directly in the repository as:

- `label_dataset_voc/`
- `label_dataset_voc_pd/`

The appendix in the thesis only presents representative samples, annotation examples,
and brief structural descriptions. This folder provides a compact explanation so a
reader can quickly understand how the complete dataset is organized in the repository.

Recommended reading order:

1. `dataset_description.md`
2. `dataset_structure.md`
3. `label_dataset_voc/`
4. `label_dataset_voc_pd/`
